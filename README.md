# Multimodal_ai_assistant
Multimodal AI Assistant (Text | Image | Audio)

A hands-on **Multimodal AI Assistant** built using **Google Gemini, LangChain, and Streamlit** that can:

- Answer questions via **text**
- Generate **images from text prompts**
- Analyze and understand **uploaded images**
- Work with **audio data** (analysis & transcription support)

This project focuses on **practical AI building** without forced UI paywalls or feature locks.
